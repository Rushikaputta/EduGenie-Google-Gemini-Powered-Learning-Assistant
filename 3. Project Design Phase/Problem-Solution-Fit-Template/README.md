## Value Proposition Canvas

### Customer Profile (Student)
**Jobs to be done:**
- Study and prepare for exams
- Understand new topics quickly
- Test their own knowledge
- Get answers to specific doubts

**Pains:**
- No access to tutors at night
- Expensive coaching classes
- Generic content doesn't match their level
- Hard to know where to start

**Gains:**
- Save time on studying
- Score better in exams
- Feel confident about topics
- Learn at their own pace

---

### Value Map (EduGenie)
**Products & Services:**
- Web-based AI learning assistant
- 5 AI-powered educational modules

**Pain Relievers:**
- 24/7 availability (no tutor needed)
- Free to use (only needs API key)
- Beginner-friendly explanations
- Clear, structured learning paths

**Gain Creators:**
- Instant summaries save hours of reading
- Quizzes boost exam confidence
- Personalized paths = organized learning
- Q&A = on-demand tutoring
