## Feature Walkthrough

### 📝 1. Topic Summarization
- **Input:** Enter any topic (e.g., "Machine Learning")
- **Output:** A concise, structured summary powered by Gemini
- **Use case:** Quick understanding of any subject

### ❓ 2. Quiz Generation
- **Input:** Topic + number of questions
- **Output:** Multiple-choice quiz with correct answers highlighted
- **Use case:** Self-assessment and exam preparation

### 🗺️ 3. Learning Path
- **Input:** Topic + preferred duration (e.g., "4 weeks")
- **Output:** Week-by-week structured study plan
- **Use case:** Organized, goal-oriented learning

### 💬 4. Q&A Module
- **Input:** Topic + specific question
- **Output:** Detailed, accurate answer from Gemini
- **Use case:** Instant doubt resolution

### 💡 5. Concept Explanation
- **Input:** Any concept or term
- **Output:** Beginner-friendly, simplified explanation
- **Use case:** Understanding complex jargon
